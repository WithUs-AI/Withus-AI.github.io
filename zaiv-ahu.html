ZAIV-AHU

ZAiV-AHU는 USB 3.2 Type-C Interface 버전의 고성능 Edge AI Accelerator 보드입니다.

최대 26 TOPS(Tera-Operations Per Second)의 고성능을 갖춘 Hailo-8™ 엣지 프로세서가 탑재되어 있습니다.다른 엣지 프로세서에 비해 성능은 높지만 크기가 작고 작동 온도가 낮은 것이 큰 장점입니다.

Specification

  • Processor: Hailo-8™ Edge Processor

  • Memory: Integreated Memory

  • TOPS: 26

  • Input Power: DC 5V/3A

  • Connectivity

    • USB Type-C

      • Power & Data(USB3.2 Gen1 x1 (5Gbps))

      • Auxiliary Power

  • Operation Temperature: 0℃ ~ 70℃

    Docker 설치

    1. Terminal 실행

    2. APT repository 업데이트

      sudo apt-get update

    3. Docker's official GPG key 추가

      sudo apt-get install ca-certificates curl gnupg sudo install -m 0755 -d /etc/apt/keyrings curl -fsSL https://download.docker.com/linux/ubuntu/gpg | sudo gpg --dearmor -o /etc/apt/keyrings/docker.gpg sudo chmod a+r /etc/apt/keyrings/docker.gpg

    4. Docker repository를 APT sources에 추가

      echo "deb [arch="$(dpkg --print-architecture)" signed-by=/etc/apt/keyrings/docker.gpg] https://download.docker.com/linux/ubuntu "$(. /etc/os-release && echo "$VERSION_CODENAME")" stable" | sudo tee /etc/apt/sources.list.d/docker.list > /dev/null sudo apt-get update

    5. Docker 설치 명령어

      sudo apt-get install docker-ce docker-ce-cli containerd.io docker-buildx-plugin docker-compose-plugin

    6. Docker 테스트

      sudo docker images

    Docker 환경 설정

    1. Terminal 실행

    2. Docker 권한 부여

              1. 네트워크 대역 설정

                Board는 기본 네트워크 설정이 아래와 같이 설정되어 출하됩니다.

                • Netmask: 255.255.255.0

                예를 들어 ZAiV의 IP가 "192.168.30.11"인 경우, 마지막 숫자를 1~254 중 Hailo가 사용중인 11을 제외한 숫자로 설정하고, Netmask를 "255.255.255.0", Gateway를 "255.255.255.0"으로 설정합니다.

              3. ZAiV IP 변경하기

                ZAiV Board의 IP를 수정하는 방법은 다음과 같습니다.

                • config.json 파일 읽어오기

                hailortcli fw-config read --output-file --ip 192.168.30.11

                • 텍스트 편집기를 이용하여 내용 수정하기

                { "network": { "should_use_dhcp": false, "static_ip_address": { "value": "192.168.30.11" //고정 아이피 설정 }, "static_gw_address": { "value": "192.168.30.1" //게이트웨이 설정 }, "static_netmask": { "value": "255.255.255.0" //넷 마스크 설정 }, "rx_pause_frames_enable": true }, "logger": { "send_via_pci": false, "send_via_uart": true } }

                주석을 참고하여 원하는 네트워크 설정으로 변경합니다.

                • config.json 파일 쓰기

                hailortcli fw-config write config.json --ip 192.168.30.11

                • ZAiV 설정

                  • Zaivrun.launch 파일 열기

                    cd ~/catkin_ws/src vi ./hailo_ai/launch/zaivrun.launch

                  • zaivrun.launch 파일 수정

                    아래 항목들이 수정됩니다. 각 값을 현재 설정된 환경에 따라 수정합니다.

                    • CamNum: Camera의 Index number

                    • CamWidth: usb_cam 노드로부터 입력받을 가로해상도

                    • CamHeight: usb_cam 노드로부터 입력받을 세로해상도

                    • InterfaceName: ZAiV 보드가 연결된 Ethernet USB의 interface 명(ifconfig로 확인가능)

                    • HEF_File: ZAiV 보드 추론에 사용할 모델의 hef파일명

              2. 실행

                source ~/catkin_ws/devel/setup.bash roslaunch usb_cam usb_cam.launch roslaunch src/zaivrun.launch
