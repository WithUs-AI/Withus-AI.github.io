DFC 환경구성

DFC 환경구성

DFC는 DataFlow Compiler의 약어입니다. 모델을 hailo 칩에서 인식 가능하도록 변환하는 역할을 수행합니다.

DFC 시스템 요구사항

1.Ubuntu 20.04, 64bit  (WSL 가능)
2.>=16GB RAM
3.python3.6~3.8 선택, pip와 virtualenv가 포함된.
4.필요한 패키지 : 선택한 파이썬 버전의 -dev 설치 ex) python3.8-dev, python3-tk, graphviz, libgraphviz-dev
5.GPU 스펙
- Nvidia’s Pascal/Turing/Ampere GPU architecture (such as Titan X Pascal, GTX 1080 Ti, RTX 2080 Ti, or RTX A4000)
- GPU driver version 470
- CUDA 11.2

설치 / 실행

설치는 Virtualenv 라는 파이썬 가상환경을 이용합니다.

  • 가상환경 생성 아래 명령어를 통해 가상환경을 생성합니다.
      virtualenv [ForderName]
  • 가상환경 로그인 아래 명령어를 입력하여 생성한 가상환경에 로그인하면 아래 사진과 같이 앞에 폴더명이 표시됩니다.
      .[ForderName]/bin/activate

  • 사전구성환경 설치 로그인 후 아래 명령어를 입력하면 가상환경에 미리 구성된 환경이 설치됩니다.

      pip install hailo_dataflow_compiler-<version>-py3-none-linux_x86_64.whl

  • 설치 확인 아래 명령어를 이용하여 명령어 수행이 잘 이행되면 설치가 잘 된 것입니다.

      Hailo -h

  • 접속 확인 아래 명령어를 사용하여 주피터 노트북을 실행할 수 있으며 같은 네트워크상에서 원격으로 접속이 가능합니다.

      hailo tutorial –ip=0.0.0.0

  • 튜토리얼 확인 위 사진의 표시된 경로를 이용하면 됩니다. 실행된 주피터 노트북을 확인해보면 0~6까지의 튜토리얼이 준비되어 있습니다.
