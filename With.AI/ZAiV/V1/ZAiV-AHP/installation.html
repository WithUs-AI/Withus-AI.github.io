installation

설치

준비

  • ZAiV-AHP Board
  • AP Board (Prepare one of the list)
    • Raspberry Pi CM4
    • Raxda CM5 (To be supported)
  • 5V/3A USB Type-C Power Adapter
  • Micro SD Card (32GB or more)
  • USB Webcam
  • 모니터 (HdMI 지원)
  • USB 2.0 통합 마우스/키보드 (통합 마우스/키보드가 없는 경우, USB 2.0 허브를 준비합니다.)
  • 케이블류
    • Micro HDMI to HDMI Cable
    • Ethernet Cable
    • USB to Micro 5Pin Cable

외부 인터페이스

  1. Raspberry PI CM4 or Radxa CM5 Modlue Slot
  2. 5V/3A Power (USB Type-C Connector)
  3. Micro HDMI
  4. USB 2.0 x 2
  5. Gigabit Ethernet (RJ45)
  6. USB to UART (micro 5pin)
  7. IO Interfaces (2.54mm Pitch 2x20 Header)
  8. External Fan (SMW200-02 Connector)
    • Pin 1 : 5V
    • Pin 2 : GND
  9. nRPI BOOT & USB_OTG
    • Pin 1 : nRPIBOOT (CM4_BOOT)
      • A low on this pin forces booting from an "RPI server" (PC or another RPi);
      • if not used, leave floating.
      • Internally 10K pull up to 3.3V
    • Pin 2 : USB_OTG_ID (but the correct OS driver also needs to be used.)
      • Input(3.3V) USB OTG Pin.
      • Internally pulled up.
      • When grounded the CM4 becomes a USB host
  10. Micro SD Slot

기본 연결

  • Raspberry Pi 이미지가 설치된 Micro SD카드를 Micro SD카드 슬롯에 삽입합니다.
  • 모니터의 전원을 연결하고 모니터와 ZAiV-AHP간에 micro HDMI to HDMI 케이블을 연결합니다.
  • USB 키보드/마우스를 연결합니다.
  • 인터넷이 가능한 Etherent Cable을 RJ45 포트에 연결합니다.
  • USB Webcam을 연결합니다.
  • UART를 통한 디버깅이 필요한 경우, Micro USB 케이블을 연결합니다.
  • 마지막으로 USB Type-C Power Adapter를 연결합니다.
