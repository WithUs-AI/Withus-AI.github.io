YoloV5 Training Env

YoloV5 Training Envirnment for Ubuntu

Anaconda 설치

  • 의존성 라이브러리 설치 실행된 가상환경에서 Yolov5 모델을 커스텀 데이터셋으로 재학습하기위해 필요한 파일을 구성합니다. 필요한 라이브러리 먼저 설치를 진행합니다.
$ sudo apt-get install Net-tools git libxcb-xinerama0
  • 아나콘다 설치 설치가 완료되면 가이드에 포함된 파일을 가상환경의 바탕화면(~/Desktop$)으로 파일을 이동합니다.
# Anaconda3-2021.05-Linux-x86_64.sh
# yolov5_retraining.ipynb

두 파일을 가상머신에 옮기고 해당 경로 (~/Desktop$) 에서 설치 스크립트를 실행하여 설치합니다.

$ sudo bash Anaconda3-2021.05-Linux-x86_64.sh
  • 아나콘다 설정
# conda 기본 경로(/root/anaconda3/bin/activate)이며 변경한 경우 변경된 path로 지정해야 합니다.
source /root/anaconda3/bin/activate
conda init
source ~/.bashrc
# conda 환경을 자동으로 활성화하는 옵션입니다.
conda config –-set auto_activate_base {True or False}
conda create -n yolov5 python=3.9
conda activate yolov5
conda install jupyter
# 워크스페이스 생성
mkdir yolov5_workspace
  • 파일 복사

생성한 Yolov5_workspce 폴더에 아래 파일을 복사합니다.

# yolov5_retraining.ipynb
# data.yaml
  • Jupyter 실행
cd yolov5_workspace
Jupyter notebook
