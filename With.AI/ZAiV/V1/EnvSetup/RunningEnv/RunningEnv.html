1.2. Running

Running

실행(추론) 가능 환경

  • X86/AMD64

    • Ubuntu 18.04, 20.04 - USB/Ethernet Interface

  • Raspberry PI Model B

    • RasbianOS - USB/Ethernet Interface

  • Raspberry PI CM4

    • RasbianOS - PCI Interface

  • RockChip RK3588 CM5 (To Be Supported)

    • Ubuntu - PCI Interface

  • Jetson Nano

    • Ubuntu 18.04 - USB/Ethernet Interface
