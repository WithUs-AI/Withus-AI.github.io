1.2.1. YoloV5 Training Env

YoloV5 Training

Docker를 이용해 Training하는 방법을 설명합니다. 사전에 Docker의 설치가 필요합니다.

  • 파일 준비 https://github.com/WithUs-AI/yolov5-docker-build 에 접속하여 Dockerfile을 다운로드 합니다.

  • 이미지 빌드 도커 빌드 명령어를 이용하여 도커 파일을 이미지로 빌드합니다. 빌드 시 "-t" 속성으로 이미지 태그를 지정합니다.

    sudo docker build --build-arg timezone=`cat /etc/timezone` -t yolov5:v0 .

  • 컨테이너 생성 이미지가 생성 후 도커 실행 명령어를 이용하여 컨테이너를 생성합니다.

    docker run --name yolov5 -it --gpus all --ipc=host -v \
/home/withus/Work/yolo_docker:/path/to/local/data/dir yolov5:v0
    • --gpus all : gpu를 사용하기위한 속성값
    • --ipc=host : 메모리 제한을 해제하기위한 속성값
    • -v [host경로]:[docker 내부경로] : 도커 컨테이너와 hostPC간의 볼륨 공유를 위한 지정
    • -it : 자동으로 컨테이너 환경이 실행
    • 마지막은 이미지 태그 지정.

  • 현재 실행 명령어에 -it 옵션이 있기 때문에 자동으로 컨테이너 환경이 실행됩니다.

    • 나가기 : ctrl + p, q
    • 종료하기 : exit
    • 다시 들어가기 : docker exec -it yolov5 /bin/bash – yolov5는 [컨테이너 이름]

  • 모델 학습하기 실행된 컨테이너 환경에서 아래 명령어를 통해 학습합니다.

    python3 python3 train.py --img 512 --data coco128.yaml --weights yolov5n.pt --cfg yolov5n.yaml

  • 모델을 onnx로 내보내기

    python3 export.py --weights runs/train/exp/weights/best.pt --img 512 --batch 16 --include onnx

  • 파일 복사 Onnx로 변환된 파일은 공유한 볼륨 경로로 복사합니다.

    cp runs/train/exp/weights/best.onnx /path/to/local/data/dir/

    이 후 Host PC의 경로에서 확인이 가능합니다.

    /home/withus/Work/yolo_docker/best.onnx
