YOLOv5

Data Flow Compile for YoloV5

  • zaiv_hailo_dfc.zip 파일을 압축해제
unzip zaiv_hailo_dfc.zip
  • 디렉토리 내용 수정
1. yolov5_dfc.py – dfc 작업을 위한 Python Script. 내용 수정 필요(경로 / 이미지 포맷 / 모델이름 등)
2. Dataset 폴더 – calibration용 이미지 폴더
3. har, hef, svg 폴더 – DFC 산출물 저장 폴더
4. xxx.onnx - 변환할 onnx파일
5. nms_config_yolov5.json - nms를 위한 하이퍼파라미터 설정
6. ethernet.alls - ZAiV USB/Ethernet Version을 위한 alls
7. pci.alls - ZAiV PCI Version을 위한 alls
  • 도커 경로로 디렉토리 복사
docker cp zaiv_hailo_dfc/ hailo_sw_suite_2023-07.1_container:/local/workspace/
  • 도커 컨테이너 터미널에서 yolov5_dfc.py 실행
python3 yolov5_dfc.py
