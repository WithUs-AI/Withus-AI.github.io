1. ROS (Noetic) 빌드 및 설치

Ubuntu 20.04 기준 ROS 설치를 가이드 합니다. 참고로 Ubuntu 20.04를 대상으로 하는 ROS Noetic Ninjemys는 최신 13번째 ROS LTS 배포판입니다. 이는 또한 마지막 ROS 1 릴리스이며, 2025년 5월까지 지원될 예정입니다.

1.1. Check

20.04인지 확인합니다.

uname -a

1.2. ROS Noetic 저장소 설정

ROS 저장소를 OS에 추가.

echo "deb http://packages.ros.org/ros/ubuntu focal main" | sudo tee /etc/apt/sources.list.d/ros-focal.list

# 출력
deb http://packages.ros.org/ros/ubuntu focal main.

1.3. official ROS key 추가.

sudo apt-key adv --keyserver 'hkp://keyserver.ubuntu.com:80' --recv-key C1CF6E31E6BADE8868B172B4F42ED6FBAB17C654

1.4. ROS 패키지 인덱스 업데이트

sudo apt update

1.5. ROS Noetic 패키지 설치

공식적으로 문서화된 4개의 ROS Noetic 메타패키지는 다음과 같습니다

대부분의 경우 전체 ROS 경험을 위해 ros-noetic-desktop-full의 설치할 것입니다. 만약 GUI가 필요없고, 로봇에 설치하는 경우 ros-noetic-ros-base를 설치하면 됩니다.

  • ros-noetic-desktop-full

    이 패키지 에는 ros-noetic-desktop 내의 모든 패키지와 인식(ros-noetic-perception), 시뮬레이션(ros-noetic-simulators) 패키지들이 포함되어 있습니다.

      sudo apt install ros-noetic-desktop-full

  • ros-noetic-desktop

    ros-noetic-desktop은 ros-noetic-desktop-full의 축소버전입니다. 대부분의 튜토리얼패키지와 아래 패키지들이 이 포함되어있습니다.

    • robot
    • vis
    • angles
    • common_tutorials
    • geometry_tutorials
    • joint_state_publisher_gui
    • ros_tutorials
    • roslint
    • urdf_tutorial
    • visualization_tutorials
      sudo apt install ros-noetic-desktop

  • ros-noetic-ros-base

    기본 ros package만 설치할 경우 선택합니다. ros_core에서 확장하여 actionlib, dynamic reconfigure, nodelets, and pluginlib와 같은 basic non-robot tools들을 포함합니다.

      sudo apt install ros-noetic-base

  • ros-noetic-ros-core

    ros-noetic-core는 아래 리스트와 같이 최소한의 핵심 패키지만 포함합니다.

    • catkin
    • class_loader
    • cmake_modules
    • common_msgs
    • gencpp
    • geneus
    • genlisp
    • genmsg
    • gennodejs
    • genpy
    • message_generation
    • message_runtime
    • pluginlib
    • ros
    • ros_comm
    • rosbag_migration_rule
    • rosconsole
    • rosconsole_bridge
    • roscpp_core
    • rosgraph_msgs
    • roslisp
    • rospack
    • std_msgs
    • std_srvs
      sudo apt install ros-noetic-core

1.6. ROS Noetic 환경설정

ROS 터미널 명령을 사용하고 catkin 디렉터리에서 헤더 파일과 같은 ROS 프로그램 파일을 찾기위해 apt install 후에 아래 명령을 실행하여 디렉터리를 포함하도록 합니다.


source /opt/ros/noetic/setup.bash

# 부팅 시 매번 해줄 수는 없으므로 rc파일에 등록합니다.

# Bash Shell의 경우
echo "source /opt/ros/noetic/setup.bash" >> ~/.bashrc


# zsh의 경우
echo "source /opt/ros/noetic/setup.zsh" >> ~/.zshrc
