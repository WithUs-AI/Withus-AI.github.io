ZAiV-AHR

ZAiV-AHR

Overview

This is the documentation for ZAiV-AHR.

ZAiV-AHR is an Edge AI Accelerator board with high performance.

It is equipped with the Hailo-8™ Edge Processor with high performance of up to 26 TOPS (Tera-Operations Per Second).

It has high performance compared to other edge processors, but it is characterized by a small size and low operating temperature.

In addition, it is composed of USB interface and can support from PC to embedded boards such as RaspberryPi and i.MX.

In particular, it is manufactured in a size optimized for Raspberry PI, and has many advantages in miniaturization and light weight.

Specifications

  • Processor: Hailo-8™ Edge Processor
  • Memory: Integreated Memory
  • TOPS: 26
  • Input Power: DC 5V/3A
  • Interfaces
    • 2 x USB Type-C
      • Power
      • Auxiliary Power
    • 1 x RJ45(Gigabit Ethernet)
    • 2 x USB 2.0
  • Communication Interface: USB3.2 Gen1 x1 (5Gbps)
  • Operation Temperature: 0℃ ~ 70℃
