installation

설치

준비

  • ZAiV-AHR Board
  • Supported Device (지원 가능 장치 중 1개 준비)
    • X86/AMD64 기반 PC
    • Raspberry Pi4 B+
    • Jetson nano
  • Micro SD Card (이미지 제작 필요 시 사용, 32GB or more)
  • USB Webcam
  • 모니터 (지원 장치에 따른 인터페이스의 모니터)
  • USB 2.0 통합 마우스/키보드 (필요시, USB 2.0 허브를 준비합니다.)
  • USB 3.0 Micro SD Reader (이미지 제작 필요 시 사용. USB 2.0도 가능.)
  • 케이블류
    • Micro HDMI to HDMI Cable (라즈베리파이4의 경우 필요.)
    • HDMI Cable (PC 및 Jetson nano의 경우 필요.)
    • Ethernet Cable
    • USB to Micro 5Pin Cable

외부 인터페이스

  1. 5V/3A Power (USB Type-C Connector)
  2. Extention Power (USB Type-C Connector)
  3. Eternet Interface (RJ45 Connector)

이미지 Flashing (이미지가 없는 경우.)

Withus 제공 이미지 사용

  • Raspberry PI CM4
    • RaspBerry PI Imager Download (설치프로그램에서 이미지를 다운로드하고 설치함. 2023.09.25 기준 1.7.5 버전)
      • Windows: https://downloads.raspberrypi.org/imager/imager_latest.exe
      • MacOS: https://downloads.raspberrypi.org/imager/imager_latest.dmg
      • Ubuntu for X86: https://downloads.raspberrypi.org/imager/imager_latest_amd64.deb
    • 사전 설치 이미지 Download:
    • 이미지 Flashing
      • Raspberry PI Imager 실행
      • 운영체제 버튼 클릭 > 사용자 정의 사용 > 파일 선택 다이얼로그에서 다운로드받은 이미지 선택
      • 저장소 버튼 클릭 > PC에 연결한 Micro SD카드 volume을 선택합니다.
      • 쓰기
  • Raxda CM5 (To be supported)

새로 설치

  • X86/AMD64

Ubuntu 18.04 버전을 설치합니다. 설치가이드는

* https://ubuntu.com/tutorials/install-ubuntu-desktop#1-overview

를 참조합니다.

  • Raspberry PI CM4

    • 이미지 준비

      • RaspBerry PI Imager (설치프로그램에서 이미지를 다운로드하고 설치함. 2023.09.25 기준 1.7.5 버전)

        • Windows: https://downloads.raspberrypi.org/imager/imager_latest.exe
        • MacOS: https://downloads.raspberrypi.org/imager/imager_latest.dmg
        • Ubuntu for X86: https://downloads.raspberrypi.org/imager/imager_latest_amd64.deb

      • ROS1에 사용된 Buster (ROS사용 시 필요)

        • http://downloads.raspberrypi.org/raspios_arm64/images/raspios_arm64-2021-05-28/ 접속
        • 2021-05-07-raspios-buster-arm64.zip 다운로드

    • Raspberry PI Imager 설치 설치프로그램의 안내에 따라 설치합니다.

    • Micro SD 연결. MicroSD카드를 Reader기에 결합 후 PC에 연결합니다.

    • 이미지 Flashing

      • Raspberry PI Imager 실행
      • 운영체제 버튼 클릭 > Raspberry Pi OS (other) > Raspberry PI OS (64-bit) 선택
      • 저장소 버튼 클릭 > PC에 연결한 Micro SD카드 volume을 선택합니다.
      • 쓰기

기본 연결 및 부팅

  • 지원가능 장치 리스트를 확인하여 장치와 ZAiV-AHR 보드 간에 Ethernet Cable을 연결합니다.
  • 보드의 경우, 이미지가 설치된 Micro SD카드를 Micro SD카드 슬롯에 삽입합니다.
  • 모니터의 전원을 연결하고 모니터와 지원 장치간에 HDMI 케이블을 연결합니다.
  • USB 키보드/마우스를 연결합니다.
  • 인터넷이 필요한 경우 wifi등으로 인터넷을 연결합니다.
  • USB Webcam을 연결합니다.
  • 마지막으로 USB Type-C Power Adapter를 연결하여 부팅합니다.
