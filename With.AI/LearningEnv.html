학습 환경 구성

Anaconda 설치하기

로컬PC에서 모델을 학습시키기 위해 환경을 구성합니다.

  1. Anaconda 설치 아래 경로에서 아나콘다(Anaconda)를 다운로드하여 설치합니다. https://www.anaconda.com/ [TODO] 설치 이미지 삽입 아나콘다(Anaconda) 설치가 완료되면 시작 화면에 폴더가 생성됩니다. [TODO] 시작화면 이미지 삽입

  2. 환경생성

  3. Anaconda Prompt를 실행하여 나온 콘솔환경에서 아래 명령어를 통해 Yolov5를 실행할 새로운 환경을 생성합니다.
      conda create --name yolo5(이름 변경 가능) python=3.9
    [TODO] create image 삽입
  • 생성된 환경 확인
      conda env list
  • 생성된 환경 활성화
      conda activate yolo5(이름 변경 가능)

환경 실행

  • 디렉터리 이동 환경 셋팅이 완료되면 cd 커맨드를 이용하여 작업을 진행할 디렉토리로 이동합니다.

  • 파일 복사 작업 디렉토리로 yolov5s_relearning.ipynb 파일 복사

  • 주피터 노트북 환경 설치 (초기 설정 시 실행)

      conda install jupyter

  • 주피터 노트북 환경 실행 아래 명령어를 입력하여 주피터 노트북을 실행하면 합니다.

      jupyter notebook
    • 주피터 노트북 환경 실행 확인 정상적으로 동작하면 아래 그림과 같이 출력되며 노란색 박스 안의 URL을 통해 웹브라우저 상의 주피터 노트북으로 접속됩니다.
