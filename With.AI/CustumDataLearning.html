Custum Data 학습

Custum Data 학습

학습 준비

  • ipython notebook 파일 실행 주피터 노트북 접속 후 주피터 노트북의 파일시스템 UI 에서 yolov5s_relearning.ipynb 파일을 찾아 실행합니다.

  • Image Labeling Program 설치하기 노트북 파일의 첫 코드는 이미지 라벨링 프로그램을 설치하는 라인입니다.

    • YOLOv5 설치하기 두번째 코드는 깃서버에 접속하여 yolov5를 설치하는 라인입니다.

  • YOLOv5 종속성 파일 설치 세번째 코드는 git으로 설치한 yolov5폴더로 이동하여 requirements.txt파일로 종속성 파일들을 설치하고 필요한 패키지들을 import하는 라인입니다.

    • Image Folder 생성 4번째 코드는 이미지들을 관리할 폴더를 생성하는 과정입니다.

Image Labeling 작업하기

5번째 코드가 이미지 라벨링을 하는 작업입니다. 해당 라인이 실행되면 아래 같은 창이 출력됩니다.

* 이미지 이동

학습할 이미지 파일들을 이전에 생성한 TrainDataset 폴더의 train 폴더의 images 폴더로 옮깁니다.

* 이미지 열기

Open Dir로 train폴더의 Images를 열어 이미지들을 한번에 열고 Change Save Dir을 같은 경로의 labels로 지정합니다.

* 객체 지정 및 라벨링

이미지를 열어 박스를 그리면 라벨이름을 지정하는 창이 출력되며 라벨이름은 저장되어 선택 리스트에 표시됩니다. 라벨작업을 할 때 마다 저장합니다. 

* YOLO5 CUSTOM 모델 생성하기

라벨링한 이미지를 가지고 Yolov5에 적용하기 위해 SaveDir로 지정한 폴더에서 classes.txt의 내용을 참고하여 첨부된 data.yaml 파일을 수정합니다.

    * Name: 에 클래스 이름을 순서대로 적고 nc: 에는 클래스의 개수를 적습니다.
    * Train과 Val은 교육용 학습데이터[\TrainDataset\test]와 확인용 학습데이터[\TrainDataset\valid]의 경로를 지정해주면 됩니다.

학습

  • 라벨링 작업 및 파일 정리가 완료되면 train.py 를 사용하는 라인이 학습을 진행하는 코드입니다.
  • {dataset.location}/data.yaml 부분을 data.yaml 경로로 지정해줍니다.
  • Yolov5의 이미지 크기는 640*640이며 batch 사이즈를 지정하고 학습횟수인 epochs를 지정합니다.
  • 데이터 라벨링을 완료하고 수정한 data.yaml 경로를 –data 옵션에 지정하고 --weights에는 학습 가중치파일인 yolov5m.pt파일로 적용하여 학습을 시작합니다.
  • 학습이 모두 완료되면 Best 가중치파일(best.pt)이 runs\train\exp 경로에 생성됩니다.
  • 학습 완료된 가중치 파일을 이용하여 tanserboard 나 detect.py로 학습된 모델의 성능과 실제 결과를 확인해볼 수 있습니다.

ONNX 변환하기

학습이 완료된 모델은 export.py를 이용하여 onnx로 변환할 수 있습니다.
